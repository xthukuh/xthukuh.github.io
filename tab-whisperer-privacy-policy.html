Privacy Policy

Effective Date: June 16, 2025

Overview

Tab Whisperer is a Google Chrome Extension that provides AI-based assistance by analyzing content on your active browser tab using Google's Gemini API.

Data Collection

Tab Whisperer does not collect or transmit any personal user data.

  • No personal information is collected or stored by the extension.
  • No data is transmitted to any server other than the Google Gemini API, which is used solely to generate responses based on the content you choose to send.
  • All user interactions are initiated manually by the user and occur locally in the browser.

Third-Party Services

Tab Whisperer connects directly to Google's Gemini API. All communication is between your browser and Google's servers using your personal Gemini API key.

Google Gemini API

  • The extension uses your API key to authenticate with Gemini services.
  • Only the content explicitly sent to the Gemini API is transmitted; no background or passive data collection occurs.
  • Refer to Googleâ€™s Privacy Policy for how it handles data.

Local Storage

Your API key is stored in your browser's localStorage to avoid repeated input. This data:

  • Is stored locally on your device only.
  • Is not synced to any server.
  • Can be cleared at any time from your browser settings.

Permissions

Tab Whisperer may request access to read the content of the current active tab. This access is used exclusively for generating contextual AI responses and is not logged or transmitted elsewhere.
